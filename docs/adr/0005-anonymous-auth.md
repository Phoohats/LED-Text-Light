# Use Firebase Anonymous Auth

ผู้ใช้ทุกคนได้ identity ผ่าน Firebase **Anonymous Auth** อัตโนมัติ ไม่มีหน้าสมัคร/ล็อกอิน

## เหตุผล
- แฟนคลับในคอนเสิร์ตต้องการเปิดแล้วใช้ทันที — การบังคับสมัครคือ friction ที่ทำให้เลิกใช้
- ยังได้ uid ที่เสถียรพอจะระบุเจ้าของ Show/preset และเขียนกฎความปลอดภัยได้

## Consequences
- หากต้องการให้ preset ติดตามข้ามเครื่องในอนาคต ค่อย "upgrade" anonymous → ล็อกอินจริงได้
- ข้อมูลผูกกับ uid anonymous อาจหายถ้าผู้ใช้ล้างข้อมูลเบราว์เซอร์ (ยอมรับได้ในขอบเขตนี้)
